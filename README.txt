
Advertising Performer
Advertising Performer is a full-stack application built with Spring Boot for the backend and React with TypeScript for the frontend.
 It manages and analyzes advertising metrics through RESTful APIs and provides a user-friendly interface for interacting with the data.

Technologies Used
Backend (Spring Boot)
Java: Programming language for backend development.
Spring Boot: Framework for creating standalone, production-grade Spring-based applications.
Spring MVC: Model-View-Controller architecture for web applications.
Spring Data JPA: Simplifies implementation of JPA repositories.
Hibernate: ORM library for mapping objects to relational databases.
PostgreSQL: Open-source relational database management system.
Lombok: Java library for reducing boilerplate code.
JUnit 5, Mockito: Testing frameworks for Java applications.
Maven: Build automation tool for Java projects.
Git: Version control system.

Backend Modules
AdController: REST API controller handling endpoints for advertising metrics.
AdMetricsRepository: JPA repository interface for database operations.
AdDataTable: Entity class representing advertising metrics stored in PostgreSQL.
AdvertisingPerformerApplication: Main Spring Boot application class.
